## 🎯 핵심 철학

```kotlin
val spec = parseProjectSpec(input).getOrElse { return failure(it) }           // 1줄: 안전한 파싱
val results = runDAGWorkflow(generators, spec)                                 // 2줄: DAG 기반 병렬 실행  
return ProjectCombiner.combineWithPartialSuccess(results)                     // 3줄: 부분 실패 허용 조립
```

## 🏛️ 아키텍처

```
app              # DAG 오케스트레이터 + 에러 관리
 ↓
generators       # 병렬 생성기 (Result 기반)
 ↓  
core            # DAG 엔진 + topological sort
 ↓
common          # Result 타입 + 기반 구조
```
